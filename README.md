# Conociendo Angular -- Mi primera app en Angular

## Descripción

Este proyecto corresponde a la actividad **"Conociendo Angular -- Mi
primera app en Angular"**.\
El objetivo es comprender el flujo básico de trabajo en Angular
utilizando **Angular CLI**, desde la creación de un proyecto hasta la
visualización de datos en la vista mediante **interpolación**.

------------------------------------------------------------------------

## Estructura del proyecto

### `src/app`

Contiene los componentes principales de la aplicación.

### `app.component.ts`

Archivo donde se define el componente principal. Aquí se crean las
variables y la lógica que luego se muestra en la vista.

### `app.module.ts`

Módulo principal de la aplicación. Declara los componentes y configura
la estructura base de Angular.

### `assets/`

Carpeta utilizada para almacenar recursos estáticos como imágenes.

### `environments/`

Archivos de configuración para distintos entornos (desarrollo y
producción).

------------------------------------------------------------------------

## Funcionalidades implementadas

-   Cambio del **título principal** de la aplicación.
-   Agregado de un **párrafo personal** en la vista.
-   Implementación de **interpolación** para mostrar una variable del
    componente.
-   Inclusión de una **imagen desde la carpeta `assets`**.

------------------------------------------------------------------------

## Instalación y ejecución

Clonar el repositorio:

``` bash
git clone https://github.com/AndyMartinezB/UTN-angular-tarea-1.git
```

Instalar dependencias:

``` bash
npm install
```

Ejecutar la aplicación:

``` bash
ng serve
```

Abrir en el navegador:

    http://localhost:4200
